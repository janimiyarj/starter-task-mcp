# Security Policy

## Supported Versions
This project currently supports the latest version on the `main` branch.

## Reporting a Vulnerability
Please report security issues through GitHub Security Advisories:
- Go to `Security` tab in this repository
- Click `Report a vulnerability`

If Security Advisories are unavailable, open a private report by contacting the repository owner directly and do not post exploit details publicly.

## Security Controls in this Repo
- GitHub Pages deployment over HTTPS only
- Restrictive Content Security Policy in `index.html`
- Automated secret scanning in CI
- Automated CodeQL scanning
- Dependabot updates for GitHub Actions
